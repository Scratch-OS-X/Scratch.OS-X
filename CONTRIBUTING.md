## Collaborer
Pour collaborer, donnez moi (par le biais des liens ci-dessus) le lien de votre projet scratch  que vous souhaiteriez ajouter à la prochaine version de Scratch.OS X.
<br>

## Bug
Pour signaler un bug, dites le directement sur le projet conserné après vous être assurés qu'il n'existait aucune version plus récente.
